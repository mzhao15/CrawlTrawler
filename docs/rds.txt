
Connect to AWS RDS PostgreSQL database:
    1. launch a AWS RDS instance
    2. install 'psycopgs' on each node of spark-cluster
        pip install psycopg2-binary
    3. find all the connection parameters
        master-username,
        password,
        endpoint,
        port (5432)
    4. run the following python codes
        import psycopg2
        params = {
            'database': database name,
            'user': '****',
            'password': '*****',
            'host': endpoint,
            'port': 5432
        }
        try:
            conn = psycopg2.connect(**params)
        except Exception as er:
            print(str(er))
    5. make sure security group allows visits
